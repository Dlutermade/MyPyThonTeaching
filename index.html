# PyThon 介紹

# 一種很簡潔極度方便的語言

警告

這也意味者 PyThon 省去很多其他語言的步驟，而這些步驟卻成跨語言或是真正理解 PyThon 語言的原理所形成的障礙牆

# 關於

# 我們會以 PyThon 為主，C++為輔的方式教學

不教純 PyThon 是因為希望大家能在初學語言理解基本觀念而非純粹的語法，C++主要是彌補語言基礎的不足，即 PyThon 所省略的步驟。

# 下列我們將分析第一著手學習語言的優缺點

語言 優點 缺點
C / C++ 能培育扎實底層原理 有太多技巧和知識點要貫通
C# 能培育語法的定義與基礎也是作者推薦 語言彈性不高
VB.Net 和英文最相近 C#能寫的 VB 也能，但 VB 已經正式被淘汰了 語法過多且本質過於相近導致寫法變化多端且型別很難控制
JavaScript 目前網頁唯一能跑語言，寫基礎東西很簡單 型別轉換方面過於炫泡、秀技最多的語言也是最難精通
PyThon 簡潔特性導致語法單一 純粹上語法會很簡單，但教核心原理卻難的跟鬼一樣

# 多程式語言，輸入 a => 輸出 a + 1 結果

看看就好

這邊主要是大略了解每個語言寫起來的感覺

  • C 代碼演示

步驟

  1. 匯入 c 的函式庫，stdio 是指 std(標準)io(輸入輸出)
  2. 定義程式進入點 Main()
  3. 宣告變數 a 為整數型態
  4. 將輸入賦值給 a
  5. a 賦值為 a + 1
  6. 輸出文字 a + 1 = 整數變數 1 ，整數變數 1 是 a
  7. 結束 Main 函式即結束程式
#include <stdio.h>

int main(){
    int a;
    scanf("%d", &a);
    a = a + 1;
    printf("a + 1 = %d", a);
    return 0;
}

  • C++代碼演示

步驟

  1. 匯入 c++的函式庫，io(輸入輸出)stream(流)
  2. 引入 命名空間 std，std 意思是指標準函式庫(Standard Library)
  3. 定義程式進入點 Main()
  4. 宣告變數 a 為整數型態
  5. 將輸入賦值給 a
  6. a 賦值為 a + 1
  7. 輸出文字 a + 1 = 整數變數 1 ，整數變數 1 是 a
  8. 結束 Main 函式即結束程式
#include <iostream>

using namespace std;
int main(){
    int a;
    cin >> a;
    a = a + 1;
    cout << "a + 1 = " << a;
    return 0;
}

  • C#代碼演示

步驟

  1. 匯入 System(系統)函式庫
  2. 定義命名空間(專案名稱)
  3. 定義程序的類別
  4. 定義靜態的 Main 函式(Program 類別的 Main 函式就是 C#的程式進入點)
  5. 宣告變數 a 為整數型態
  6. 將輸入行從字串轉成整數再賦值給 a
  7. a 賦值為 a + 1
  8. 輸出文字(a + 1 = ) + a 轉成字串
using System;

namespace ConsoleProject
{
    class Program
    {
        static void Main()
        {
            int a;
            a = Convert.ToInt32(Console.ReadLine());
            a = a + 1;
            Console.WriteLine("a + 1 = " + a.ToString());
        }
    }
}

  • VB.Net

步驟

  1. 匯入 System(系統)函式庫
  2. 定義模組(專案名稱)
  3. 定義靜態的 Main 子程式(模組的 Main 子程式就是 VB.Net 的程式進入點)
  4. 宣告變數 a 為整數型態
  5. 將輸入行從字串轉成整數再賦值給 a
  6. a 賦值為 a + 1
  7. 輸出文字(a + 1 = ) + a 轉成字串
Imports System;

Module ConsoleProject
    Sub Main()
        Dim a As Integer
        a = Convert.ToInt32(Console.ReadLine())
        a = a + 1
        Console.WriteLine(a)
    End Sub
End Module

  • JavaScript(JavaScript 和 Java 是兩個語言差很多不要當作同個東西切記)

步驟

  1. 宣告變數 a (JavaScript 不存在型別的宣告)
  2. a = 輸入框的字串
  3. a 型別從字串轉成數字
  4. a 賦值為 a + 1
  5. 輸出文字(a + 1 = ) + a (JavaScript 可以數字自動轉字串)
var a;
a = prompt();
a = Number(a);
a = a + 1;
console.log("a + 1 = " + a);

  • PyThon

步驟

  1. 定義 a = 輸入 (PyThon 不存在純粹宣告變數)
  2. a = a 轉整數
  3. a = a + 1
  4. 控制台輸出變數 a
  5. a 賦值為 a + 1
  6. 輸出文字(a + 1 = ) + a 轉字串
a = input()
a = int(a)
a = a + 1
print("a + 1 = " + str(a))

# 結論

說明

上述語言除了 VB.net 外幾乎都是基於 c 語言所延伸的，其中以 JavaScript 和 PyThon 最為簡潔，這是因為這兩個語言都是腳本語言即逐步執行(一行讀取跑完才跑下一行)而非編譯語言(產生執行檔)，也因此編譯語言速度也比較快，而進入語法篇章時候便會發現為啥 PyThon 是最為簡潔的語言。

若仔細閱讀還會發現 PyThon 少了變數的宣告其他都有變數的宣告這也是我們目前代碼展示所出現的 Python 步驟省略事情
