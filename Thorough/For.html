# PyThon迭代For無窮迴圈

那PyThon是迭代就沒辦法了嗎? 答:錯，只是會需要超深入知識

第一種利用List的添加元素特性，達到始終有尚未迭代的元素

這還算初階開發

a = [0]
b = 0
for i in a:
    b += 1
    a.append(b)
    print(i)

第二種 利用yield While

def Func():
    i = 0
    while True:
        i += 1
        yield i

for i in Func():
    print(i)

第三種 利用yield Recursive(遞迴)

由於是遞迴會有所謂的Call Stack上限導致程式中止

def Func(i):
    i += 1
    yield i
    yield from Func(i)

for i in Func(0):
    print(i)

# While寫PyThon的For迴圈

n = [1, 2, 3, 4, 5, 6, 7, 8]

if hasattr(n, "__iter__"):
    Temp = iter(n)
    if hasattr(Temp, "__next__"):
        while True:
            try:
                i = next(Temp)
                print(i) # Block
            except StopIteration:
                print("結束迭代")
                print(hasattr(Temp, "__getitem__"))
                break
    elif hasattr(n, "__getitem__"):
        print("這不是迭代器")
        Temp = 0
        while True:
            try:
                i = n[Temp]
                print(i) # Block
                Temp += 1
            except IndexError:
                print("結束迭代")
                break
    else:
        print("這不是迭代物件")
elif hasattr(n, "__getitem__"):
    print("這不是迭代物件，但可以遍歷")
    Temp = 0
    while True:
        try:
            i = n[Temp]
            print(i) # Block
            Temp += 1
        except IndexError:
            print("結束遍歷")
            break
else:
    print("這不是迭代物件")
