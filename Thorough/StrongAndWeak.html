# 強型別與弱型別概念

# 強型別 => 不允許值自動從A Type變成B Type 即我們要手動轉換不會自動轉換 專業術語強制轉換

# 弱型別 => 允許值自動從A Type變成B Type 專業術語隱含轉換

詳細可以參考知呼

# JavaScritp

# 江湖上最會轉的語言之一，沒人敢說專精的語言

備註因為所有型別都可以轉

# 字串和數字相加


1 + "1"
1

結果

"11"
1

1

# VB.Net

# 江湖上最反邏輯語言

# 數字和字串相加


1 + "1"
1

結果

2
1

# True + 1


True + 1
1

結果

0
1

因為VB.Net的True是-1，其他主流語言都是1

# C++

# 江湖上最隱含轉換的語言之一，但邏輯卻是最明顯的

# 數字和字元相加


1 + '1'
1

結果

50
1

因為Char是存放數值的即UTF-8碼，"1"的碼同Ascii碼即49，49 + 1 = 50

# True + 1


true + 1
1

結果

2
1

# 字串 + 數字


"111" + 10
1

結果

" "
1
