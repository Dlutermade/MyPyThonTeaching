# 深入邏輯運算操作

在程式世界中除了Not、Or、And還有Xor

必須先了解的事情

True轉成Number是1
False轉成Number是0
0轉成Boolean是False
非0轉成Boolean是True

# Not 反閘

Not是唯一必然產生True或是False的邏輯符

遇到數字先轉Boolean在作Not運算。

語法

C++ PyThon
! not

在C++世界中有一個 ~ 也是Not但是是根據Type長度轉成0和1反過來比如 短正整數 3 會變成 65533 因為短正整數最大是65535

print(not 0)
print(not -1)
print(not 1)

|| or

  1. |是Number的
  2. ||是Boolean的

# Xor 互斥或閘

Xor 是指兩邊不同成立，一樣不成立

Xor有分兩種Number的Xor，Boolean的Xor

語法上PyThon和C++通用。

語法

Boolean的 != 是利用兩邊值不同所推論出來的

Number Boolean
^ a != b

語法編沒有原因

Xor用到機會很少，且難度最高

print(12 ^ 3)
print(10 ^ 10)
print(7 ^ 2)
