# 最大公因數GCD

GCD定義為最大整除兩個數的數字

解答
  1. 窮舉法

很笨的方式但也是最直覺最直觀的方法

算法效率為 O(min(n1,n2)) 即 最小數 = 圈數

n1 = int(input())
n2 = int(input())
GCD = 0
for i in range(1, min(n1, n2) + 1):
    if n1 % i == 0 and n2 % i == 0:
        GCD = i
print(f"GCD: {GCD}")

  1. 輾轉相除法

出於 歐幾里得 所撰寫的 < 幾何原本 > 和 東漢 < 九章算術 >

是基本上遇到GCD題目必用算法

算法效率為 O(log(n)) 即 log2(n1 + n2)，當數字大的時候速度會遠比上面O(min(n1,n2)) 快上非常多，這也是為啥GCD題目很多人會超過時間

題目一

n1 = int(input())
n2 = int(input())
MaxNumber = max(n1, n2)
MinNumber = min(n1, n2)
while MinNumber != 0:
    MaxNumber , MinNumber = MinNumber , MaxNumber % MinNumber
print(f"GCD: {MaxNumber}")

  1. 輾轉相減法

出於 東漢 的 < 九章算術 > 所寫 [ 更相減損術 ] 和古希臘的畢氏音程(畢氏定理同個人)

原理同 輾轉相除法 只是除改成減

n1 = int(input())
n2 = int(input())
while n1 != n2:
    if n1 > n2:
        n1 -= n2
    else:
        n2 -= n1
print(f"GCD: {n1} 此法特色兩數相等 { n1 == n2 }")

# 費式數列(Fibonacci numbers)

  1. 公式 F(n) = F(n-1) + F(n-2)
  2. 已知 F(1) = 1，F(0) = 0
  3. 範例 F(2) = F(1) + F(0) = 1
  4. 0(第0項)，1(第1項)，1(第2項)，2(第3項)，3(第4項)，5(第5項)，8(第6項)
解答

版本1 For

F = beforeOne + beforeTwo

同等

F(n) = F(n-1) + F(n-2)

假設 F(n-1) = F(1)，F(n-2) = F(0)則F(n) = F(2)

即第一圈是F = F(2)，第二圈F = F(3)

但是呢我們第一圈i是2

即

i = 2，F(2) i = 3，F(3) 直到i = n 即F(n)

n = int(input())
beforeOne = 1
beforeTwo = 0
F = 0
if n == 0:
    F = beforeTwo
elif n == 1:
    F = beforeOne
for i in range(2,n + 1):
    F = beforeOne + beforeTwo
    beforeTwo = beforeOne
    beforeOne = F
print(f"第{n}項是 {F}")
