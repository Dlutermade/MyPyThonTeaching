# 所謂輸入和輸出會牽涉到一個概念IOStream(輸入/輸出流)

IO Stream

IO是指輸入Input和Output簡寫，而Stream概念是指使用者的行為(即輸入與輸出)到程式裡面的橋樑

以下列C++語法最為明顯

#include <iostream>

using namespace std;
int main(){
    int a;
    cin >> a;
    cout << a;
    return 0;
}
