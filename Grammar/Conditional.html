# If 陳述式

# 觀念

  1. 由上往下判斷
  2. 只會執行一個Block
  3. 條件是指True或是False，如果是字串或數值即帶bool(n)所產生的結果
  4. If 代表 如果
  5. Else 代表 否則
  6. 必定會有一組if
  7. 會有0 ~ 多組else if
  8. 會有0 ~ 1組else
  9. else 下面的else if不會執行

題目3. 分別輸入a和b和c，求 aX ^ 2 + bX + C = 0的根

  1. 第一行a，第二行b，第三行c
  2. 輸出有三種 無解，一個根，兩個根
  3. b^2 - 4ac，等於0，一個根，小於0，無解，大於0兩個根
  4. ( -b + (b^2 - 4ac) ^ 0.5) / 2a會得到最大根
  5. ( -b - (b^2 - 4ac) ^ 0.5) / 2a會得到最小根

詳細題目取自高中生解題系統

解答

版本1

a = int(input())
b = int(input())
c = int(input())
NumberOfIsRoot = b ^ 2 - 4 * a * c
if NumberOfIsRoot == 0:
    print("Two same roots x=" + str((-b) / (2 * a)))
elif NumberOfIsRoot > 0:
    print("Two different roots x1=" + str((-b + NumberOfIsRoot) ** 0.5 / (2 * a)) + "x2="+ str((-b - NumberOfIsRoot) ** 0.5 / (2 * a)))
else:
    print("No real root")

# Switch 陳述式

觀念

  1. Switch中文是交換器
  2. 代入一個值進去
  3. 尋找對應值的Case
  4. 從對應的Case往下執行
  5. Break可以跳出Switch
  6. Python沒有但主流語言都有
  7. 少用Switch，能用If就用If
  8. 存在多種case
  9. 會有0~1個default
  10. 是透過GoTo(跳躍結構)方式到指定行數往下執行
  11. 盡量不要使用GoTo功能，在現代這種功能主要是底層工程師為了效能才會用(底層是指驅動程式、韌體程式)
  12. Apcs有考過
  13. 沒裝C++的人，可以用網頁瀏覽器F12的Console寫JavaScript體驗看看

JavaScript只要把輸出改成console.log()就好

# Switch流程圖

C++寫法

switch(引數){
    case 值1:
        執行區塊1;
    case 值2:
        執行區塊2;
    default:
        執行區塊3;
}

流程

  1. 帶入一個值稱為引數
  2. 尋找對應的值
  3. 假設對應的值是值1，執行區塊1，再執行區塊2，再執行區塊3
  4. 假設對應的值是值2，執行區塊2，再執行區塊3
  5. 假設沒有對應的值，執行區塊3

switch(引數){

        break;

        執行區塊2;
        break;
    default:
        執行區塊3;
