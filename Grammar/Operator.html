# 操作符

即指對於資料(變數)的操作。

操作符也稱為運算子(Operator)，而變數和值稱為運算元(Operand)。

運算元 運算子 運算元 運算子 運算元
a = a + 10

整體稱為運算式(Expression)

# 分類

我們都有照先後順序進行撰寫。

名稱 作用
運算符 進行普通數學運算
比較符 進行比較關係產生是(True)與否(False)的值
邏輯符 透過兩者關係產生值(和比較不同)
賦值符 把值給變數

# 比較符

# 比較兩者關係，產生布林值

名稱 作用 輸入 輸出
== 進行等於判斷 10 == 10 True
!= 進行不等於判斷 10 != 10 False
< 進行小於判斷 10 < 3 False
> 進行大於判斷 10 > 2 True
<= 進行小於等於判斷 10 <= 10 True
>= 進行大於等於判斷 10 >= 3 True

字串的比較是採用UTF-8碼比較，如果只有數字和英文，是和Ascii碼一樣的


警告

兩個=叫做等於，一個=叫做賦值，不要搞混在絕大多數語言都一樣，甚至有些語言有三個=叫做型別與值都等於EX:JavaScript。


請嘗試兩個不同的型別進行比較操作符，因為同型別都可以做比較操作符，所以不需要特別做

EX:

print( 0.0 == 0)
print( True < 2.5)
print( "aa" != 0)
print( "aa" > 0)

# 賦值符

把右邊結果給左邊變數

只有變數可以賦值

變數在尚未宣告賦值時，不能使用除了賦值以外的任一賦值EX:加賦值、減賦值

原則 先右在左

以下皆用a變數當範例，輸出a，並假設a為10

名稱 輸入 全寫 輸出
賦值 a = 10 已經是全寫 10
指數賦值 a **= 2 + 1 a = a ** (2 + 1) 1000
乘賦值 a *= 2 - 3 a = a * (2 - 3) -10
除賦值 a /= 1 + 2 a = a / (1 + 2) 3.3333333333333335
整數除賦值 a //= 1 + 2 a = a // (1 + 2) 3
加賦值 a += 1 + 2 a = a + (1 + 2) 13
減賦值 a -= 1 + 2 a = a - (1 + 2) 7

括號意義

基於先右在左原則，我們必須先把右邊算完才會進行任一一種賦值， EX:
