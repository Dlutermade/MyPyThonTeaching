# 字串插值(主推)

# 有版本限制Python 3.6版以上才能用

a = 100
b = "你好"
print(f"{b}， 我要價: {a}元")
1
2
3

輸出結果是 "你好， 我要價: 10元"

這種寫法可以自動轉換型別，也很方便輸出使用

也可以這樣寫，因為他是字串的一種寫法

a = 100
b = "你好"
c = f"{b}， 我要價: {a}元"
print(c)
1
2
3
4

寫法如下

f"{內容1}文字1{內容2}文字2{內容3}" # {}內可以是數值、變數等等只要是有值的存在都可以
1

更多寫法參考

f"{1 + 2}{3 + 4}{1 + 2 + 3 + 4}" # {}內是數字並非字串，計算完後轉成字串
1

結果是 "3710"

# 字串格式化(舊版)

# 需要背符號意思

項目 功能
c 輸出字元
s 輸出字串
d 輸出整數
f 輸出浮點數

需要記得只有s、d、f

a = 100
b = "你好"
print("%s,我要價: %d元" %(b, a))
1
2
3

輸出結果是 "你好， 我要價: 10元"

這種寫法不能自動轉換型別

也可以這樣寫，因為他是字串的一種寫法

a = 100
b = "你好"
c = "%s,我要價: %d元" %(b, a)
print(c)
1
2
3
4

寫法如下

"%<功能> 內容1 %<功能>元" %(字串裡面第一個%, 字串裡面第二個%) # ()內引數必須要可以轉換成指定的功能
1

更多寫法參考

"%7.5f 和 %3.5f 和 %4d" %(3.14159265359, 3.0, 10) # 需要注意Float不允許轉換成Int，但Int可以轉成Float
1

結果是 "3.14159 和 3.00000 和 10"

因為%的功能是 %最小長度.小數點數量<功能>

  1. 字串總長度必須大於等於7且有小數點後5數
  2. 字串總長度必須大於等於3且有小數點後5數
  3. 字串總長度必須大於等於4 (仔細發現會有兩個空白在10的前面)
  4. 需要特別注意.x會導致小數四捨五入剛好，這個數字是2所以是捨去不是進位

# 字串格式化(新版)

# 不用特別背符號意思，有版本限制必須python 2.6以上，python 3則沒限制

a = 100
b = "你好"
print("{},我要價: {}元".format(b, a))
1
2
3

輸出結果是 "你好， 我要價: 10元"

這種寫法能自動轉換型別

也可以這樣寫，因為他是字串的一種寫法

a = 100
b = "你好"
c = "{},我要價: {}元".format(b, a)
print(c)
1
2
3
4

寫法如下

"{} 內容1 {}元".format(對應第一個{}, 對應第二個{})
1

更多寫法參考

"{:7.5f} 和 {:3.5f} 和 {:4d}".format(3.14159265359, 3.0, 10)
1

結果是 "3.14159 和 3.00000 和 10"

"{0:7.5f} 和 {0:3.5f} 和 {2:4d}".format(3.14159265359, 3.0, 10, 30) # 需要注意Float不允許轉換成Int，但Int可以轉成Float
1

結果是 "3.14159 和 3.14159 和 10"

在程式世界中0是第一項，1是第二項

  1. 字串總長度必須大於等於7且有小數點後5數
  2. 字串總長度必須大於等於3且有小數點後5數
  3. 字串總長度必須大於等於4 (仔細發現會有兩個空白在10的前面)
  4. 需要特別注意.x會導致小數四捨五入剛好，這個數字是2所以是捨去不是進位

# 正常用途進階寫法

# 基於input()和print()的延伸

輸出僅適用於python3，python2都不適用

  1. 輸入有提示內容
a = input("請輸入a: ")
print(f"你的輸入是 {a}")
1
2

假設我打cc則會是下列結果

請輸入a: cc
你的輸入是 cc
1
2

這種設計對於使用者來說也比較明確知道要輸入啥

  1. 輸出多筆資料
print(10, 20, 30 ,"fff" ,"40")
1

結果為"10 20 30 fff 40"

因為print會把()內的多個引數用空白合併

  1. 修改輸出結尾
print(10, end = "")
print(10, end = "")
1
2

結果為

  1. "1010"

因為結尾是空字串不是換行


預設是下列代碼

print(10, end = "\n")
print(10, end = "\n")
1
2

結果為

  1. "10"
  2. "10"

# 比賽用寫法

# 比賽會要求使用標準(std)輸入輸出所以請用以下方式

有個重點

Python是強型別這是關鍵，所以輸入都是字串還是要記得轉型別

  1. 輸入

系統.標準輸入.讀取單行

import sys # 匯入系統模組

a = sys.stdin.readline() #這行會使用標準輸入讀取到enter停下來
1
2
3

通常還會再進階一點

系統.標準輸入.讀取單行.去前後空白

因為通常讀取字串時候都會有些奇怪的空白被讀取導致輸入怪怪的

EX:換行本身就會產生一個空白

import sys # 匯入系統模組

a = sys.stdin.readline().strip() #這行會使用標準輸入讀取到enter停下來
1
2
3

系統.標準輸出.輸出文字

import sys # 匯入系統模組

sys.stdout.write("你好") #這行會輸出一段字串且不換行
