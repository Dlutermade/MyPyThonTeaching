# While觀念

  1. 在程式意思是無窮迴圈

  2. 用於不確定的東西上比如當條件成立重複甚麼事情直到怎樣

比如1.重複輸入直到輸入是0結束迴圈

比如2.重複進行登入直到登入才結束迴圈

  1. Break和Continue一樣沿用

  2. 在新手開發思維上For能作的事情While都可以作到，但是While能作到的For未必能作到(因為不討論秀技)

  3. 英文意思 當... (也就是<當><條件成立>重複執行)

# 學While的目的

聯絡對象 = 前女友
while True: # 讓條件保持始終成立
    聯絡對象.傳送訊息("我們復合好不好!?")
    if 聯絡對象.對方最新傳的訊息 == "好阿":
        告訴使用者("你前女友已經願意和你復合了!!")
        break # 結束while的重複循環
    else:
        告訴使用者("你前女友還是沒跟你複合，3秒後進行下一次訊息傳送")
        程式.暫停(3000) # 3000是指3000ms，同3秒

# Do ... While

  1. PyThon沒有，C++和JavaScript有
  2. 結束只有兩種可能性: 條件不成立、被Break
  3. 是先執行再判斷
  4. 無論如何最少都會跑一次

# Do...While流程圖

C++和JavaScript語法

do{
    Block
}while(條件);
1
2
3

C++範例

int i = 0;
do{
    cout << i << "\n";
    i += 1;
}while(i < 3);
cout << "結束的i = " << i;

我們用虛擬碼展現連線程式開發

do{
    程式.暫停(3000); // 暫停3秒後再往下執行
    連線裝置 = 取得周圍第一個可連線裝置()
    使用者介面輸出("嘗試連線中");
}while(連線裝置.連線狀態 != true); // 在尚未連線前都會嘗試重新連線
使用者介面輸出("已成功連線");
