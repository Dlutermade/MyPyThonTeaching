# 變數 Variable

# 是一種存放值(Value)的存在，可以重新賦值稱為變數，不可以重新賦值即常數

語言差異

PyThon沒有常數只有變數，而C++則都有

我們將10的Type印出來為int，int是簡寫全名叫做Integer即整數，表示非小數點的數字

print(type(10))
1

PyThon 賦值一個變數a，在PyThon世界賦值和宣告是同時進行的。

下列代碼 我們對於一個變數a，賦值(=)了值(Value)10

a = 10
print(type(a), a)
1
2

我們將Type印出來型別為int，而第二個印出的則是Value即值

print(引數1, 引數2, ...) PyThon允許多個引數以空格區分印出

引數，英文為Argument，表示傳過去(或稱帶過去)的變數或是常數

# 型別 Type

# 型別就比如工廠裡面不同的角色，有管人的人、有管錢的人、有生產的人，<各盡其責>。

在PyThon世界中，有四大型別(主要會使用的)

  1. str (字串:String)
  2. int (整數:Integer)
  3. float (浮點數:Float)
  4. bool (布林值:Boolean)

# 整數篇

先了解觀念

Python是物件導向語言，且為動態型別，這代表所有Type都是物件，所有變數都可以存放各種Type的物件， 建構這個Type的物件稱為建構子是函式的一種比如int()是int這個Type的建構子，所產生的值比如10就是物件。

請嘗試下列三組測試資料(空白是只打出空白不是複製貼上)

  1. 10
  2. -10
  3. -0
  4. 10.5
  5. a
  6. 10a
  7. 空白10空白
  8. 1空白0
a = input()
a = int(a)
print(type(a))
print(a)
1
2
3
4
輸出結果分析

結果應該是

  1. 10
  2. -10
  3. 0

這是基於不存在負的0所以顯示0

  1. invalid literal for int() with base 10: '10.5'

這是錯誤意思無法把字串"10.5"轉成Int Type

  1. invalid literal for int() with base 10: 'a'

這是錯誤意思無法把字串"a"轉成Int Type

  1. invalid literal for int() with base 10: '10a'

這是錯誤意思無法把字串"10a"轉成Int Type

  1. 10
  2. invalid literal for int() with base 10: '1 0'

這是錯誤意思無法把字串"1 0"轉成Int Type

結論分析

  1. int是先去除前後空白即" 10 "變成"10"，再轉成整數即"10"轉成10
  2. "10.5"無法轉成數字是因為"10.5"是字串不是浮點數，所以這個字串中出現的"."導致無法轉換
  3. 假設遇到"10.5"可以先轉float再轉int比如下面代碼
a = input()
a = float(a)
a = int(a)
print(type(a))
print(a)
1
2
3
4
5
  1. 觀念float代入int()函式會建構出原本float去除小數點的int。
簡寫教學 
a = int(input())
print(type(a))
print(a)
1
2
3
  1. 簡寫原理解釋，下列這行會先作input()才做int()，即()內的函式先跑(input)才跑外面的函式(int)
  2. 遇到簡寫就記得先找()最深處的再往外擴散慢慢做

# 浮點數篇

先了解觀念

浮點數是指有小數點的數字

請嘗試下列三組測試資料(空白是只打出空白不是複製貼上)

  1. 10
  2. 10.5
  3. 10.0
  4. -0
  5. 1空格0
  6. 空白10空白
  7. 10a
a = input()
a = float(a)
print(type(a))
print(a)
1
2
3
4
輸出結果分析

結果應該是

  1. 10.0
  2. 10.5
  3. 10.0
  4. -0.0
  5. could not convert string to float: '1 0'

這是錯誤意思無法把字串"1 0"轉成Float Type

  1. 10.0
  2. could not convert string to float: '10a'

這是錯誤意思無法把字串"10a"轉成Int Type

結論分析

  1. Float是先去除前後空白即" 10.5 "變成"10.5"，再轉成整數即"10.5"轉成10.5
  2. 0或是XX的數字會變成XX.0是基於浮點數有小數點的存在
  3. 假設遇到XX.0的數字想去掉小數點可以透用int()
a = input()
a = float(a)
a = int(a)
print(type(a))
print(a)
1
2
3
4
5
  1. 觀念float代入int()函式會建構出原本float去除小數點的int。
