# 程式碼結束

# 認識寫法上的不同

PyThon 單行程式碼結束必須換行是以檔案格式作為主要區分

a = 1
b = 1
1
2

C++(CPP) 單行程式碼是以;代表結束以撰寫下一段Code作為主要區分

int a = 1;int b = 1;
1

Block(區塊)

區塊裡面會存在多行程式碼，而區塊則是封裝這些片段成一體，而執行這單一的區塊則等於執行這些片段

PyThon 區塊(Block)是以Tab(四個半形空白)作為區分

if True:
    print("我是內部區塊")
print("我是外部區塊")
1
2
3

C++ 區塊(Block)是以{ }作為區分，{}內即是區塊

if (true) {cout << "我是內部區塊";} cout << "我是外部區塊";
1

往後還會提到更多差別，先從格式上認識
